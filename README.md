### 🗓 Day 9
(對應專案：[chat(chat2)] ✅、[chat(chat3)] ✅)

- 練習處理有時間戳記與說話人的聊天紀錄（如 LINE 對話）

- 處理未加空格的時間與人名（使用 [:5] 取時間、[5:] 取人名）

**Day 9 小筆記：** 
- 字串切割技巧是處理真實資料時最常見的能力之一
- 拆解格式混亂的資料時，善用 split() + 字串索引是關鍵
- 統計與轉換可以同步進行，讓程式更有效率
